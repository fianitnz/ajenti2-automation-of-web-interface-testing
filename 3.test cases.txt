1.1
Хедер (без логина)

    хедер содержит:
        Ajenti
            ссылка расположенная с левой стороны

            ведущая:
                $адрес/view/

            перенаправляющая:
                $адрес/view/dashboard

                без логина возвращающая:
                    $адрес/view/login/normal

            при наведении:
                # курсор действия
                ссылка становится контрастнее


        иконка c $hostname
            расположена с правой стороны левее

            действия отсутствуют


        кнопка развернуть\свернуть по ширене
            расположена с правой стороны правее
            иконка стрелки внуть\наружу по диагонали "/"

            при наведении:
                # курсор действия
                фон становится темнее

            при нажатии
                фон становится темнее наведения
                иконка меняет цвет на черный

            при клике
                стрелки наружу разворачивающая страницу на ширину окна
                стрелки внуть сворачивающая страницу до ~80% окна

1.3
Страница логина (логин)

    ввод в поле Username:

        ввод $username

    * переключение к полю Password с помощью клаивши Tab
    * переключение к полю Password щелчком на поле Password

    ввод в поле Password:

        ввод $password

    нажатие кнопки Log in:
        верный логин и пароль
            анимация вращения иконки замок
            замена иконки замка на открытый
            по верхней кромке хедера полоса анимация загрузки
            перенаправление $адрес/view/dashboard

        неверный логин или пароль
            анимация вращения иконки замок
            по верхней кромке хедера полоса анимация загрузки
            анимация иконки замок останавливается
            попап красного цвета с иконкой восклицательный знак на щите \
                                         и надписью Authentication failed

Хедер (с логином)

    содержит:
        кнопка меню
            при нажатии
                скрыть\показать меню

        Ajenti $menuname
            ссылка расположенная с левой стороны

            ведущая:
                $адрес/view/

            перенаправляющая:
                $адрес/view/dashboard

        иконка c $hostname
            расположена с правой стороны левее

        кнопка выпадающего меню пользователя
            расположена с правой стороны по центру
            иконка круг в круге первая буква $username
                если пользователь root верхняя правая четверть содержит \
                                            иконку двойной вствки в круге

            при нажатии
                выпадающее меню под кнопкой смещенное влево
                    #1 иконка юзера с $username правее UID $uid
                    #2 иконка двойной вставки с Elevate
                        ??? отсутствует у root
                        ??? при нажатии разлогинивает
                    #3 иконка IO с Log out
                        при нажатии
                            разлогинивает
                            попап красного цвета с иконкой восклицательный \
                                                   знак на щите и надписью \
                                                     You session has expired
                            перенаправление $адрес/view/login/normal
                                в адресной строке остается //view/dashboard \
                                это возврат к странице от куда было разлогинено

        кнопка развернуть\свернуть по ширене
            расположена с правой стороны правее
            иконка стрелки внуть\наружу по диагонали "/"

            при нажатии
                стрелки наружу разворачивающая страницу на ширину окна
                стрелки внуть сворачивающая страницу до ~80% окна
